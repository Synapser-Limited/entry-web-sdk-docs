## BACKGROUND AND PURPOSE

The Entry Web SDK is a production-ready TypeScript/React library that provides secure biometric authentication capabilities using AWS services (Amplify, Cognito, and Rekognition). This SDK enables client applications to integrate facial recognition and liveness detection for user identification and registration.

**Key SDK Capabilities:**

- Biometric user registration and identification workflows
- Active liveness detection to prevent spoofing

**Distribution Model:**
This SDK is distributed via GitHub Packages (@synapser-limited/entry-web-sdk) to authorized users/organizations who integrate it into their applications.

- **Package Registry:** [GitHub Packages](https://github.com/Synapser-Limited/entry-web-sdk/pkgs/npm/entry-web-sdk)
